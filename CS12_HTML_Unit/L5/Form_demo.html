Lesson 5 Demo: HTML Forms

A form collects information from the user. Everything inside a <form> tag is part of that form. Let's look at each piece one at a time.

1. Text Inputs & Labels

A <label> describes what an input is for. The label's for attribute must match the input's id. Try clicking the label text — notice it moves your cursor into the box!

5. Dropdown Menu (select)

A <select> creates a dropdown menu. Each choice is an <option> inside it.

6. Textarea (Multi-line Text)

A <textarea> lets users type a longer message. Use rows to set how tall it is.
